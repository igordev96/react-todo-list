## Features

- Delete task
- NewItem/Add button change dynamically


## Run Locally

Clone the project

```bash
  git clone https://github.com/igordev96/react-todo-list
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
